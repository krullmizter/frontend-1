Intro

Projektet har varit riktigt roligt och givande, jag har lärt mig mycket nytt och användbart och har nu en ny och förbättrad uppfattning om JavaScript. Jag tycker att jag både har skapat en ”webbapp” som både funktionellt och grafiskt fungerar. Jag har använt mig av det mesta som vi gått igenom på lektionerna, plus lite till. Lite extra saker som jag gjort är detta:

  • Event listener på decimaltalen när man skriver.
  • ”Hela” binära, oktala och hex tal i decimal kalkylatorn.
  • Social skydds signumet med ett flertal regex i JavaScript.
  • En ”riktigt” datum väljare med några valideringar i sig (inget minus datum).
  • Färgkonverterare som tar in allt vad man vill, och lite till.
  • Ett ”coolt” sätt att räkna ut referensnumrets, och dess sista tal.
  • En hel del input validering och error meddelanden
    • Har validerat JavaScript, CSS och HTML.
    • Använder jag bla. flex-box för snygg och responsiv design, och FontAwesome (+ annat) för att pigga upp sidan.
    • Använder mej av CSS variabler, CSS prefixes och REM units för bättre CSS.
    • Allt som allt så är sidans fil och folder struktur prima enligt mig.

Uppgfiterna

Det var bra att alla deluppgifter hörde ihop med varandra på ett visst sätt, på så sätt var det mycket lättare att skapa en helhet i projektet. Den lättaste del uppgiften var nog stränghanteringen och den svåraste var socialskyddsignum och/eller RGB färgkonverteraren. Dom andra deluppgifterna tyckte jag att var mellansvåra men man kunde snabbt och ganska enkelt komma på vad man skulle göra.
Socialskyddsignumet var svårt att få korrekt p.g.a. regex, men efter lite problemlösning med Andrej så fixade det sig. Jag inser nu att jag skulle ha behövt använda mig av en new Date() funktion för att få ut exakta datum och kunna räkna med skottår och olika ogiltiga datum i vissa månader i uppgiften, men jag tycker att jag fick det tillräckligt bra med bara regex och condotional statments.
Referensnummer uppgiften var också ganska svåra för jag hittade inte hur man gjorde internationella numror.
Färgkonverteranden var också knepig p.g.a. att jag använde först bara en .style.backgroundcolor i JavaScript, men efter att jag provade på lite så kom jag på att jag borde också få allt att fungera med RGB och RGBA, så några extra if statments och så började det fungera.

Avslut

Saker som jag skulle förbättra är nog mest ang

  • Bättre inputvalideringar
  • Fått async, form och addEventListener att fungera bättre då input fälten ändras
  • Mer och bättre RegEx
  • Fått allt bonus gjort
  • Bättre skriven, fungerande och snabbare kod
  • Snyggare layout

Jag tycker att jag nu kan JavaScript mycket bättra och det känns som att jag nu kan ta mig an dom nya projekten utan problem. Tycker kursen är rolig, skulle dock vela se att vi kan se alla projekt då kursen börjar så man få "dona" med dom redan från början. Det har också hjälpt att jag nu kan JavaScript bättre, för på min nuvarande arbetsplats så använder vi en hel del JavaScript, och det har varit ganska flummigt för mig tidigare.

Det står i clockify att jag samlat ihop ca. 23:15h, men jag skulle tippa på ca. 25h totalt.
